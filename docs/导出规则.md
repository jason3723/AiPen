## AiPen 导出 Word 的完整规则清单

### 一、页面布局

| 规则 | 默认值 | 说明 |
|---|---|---|
| 纸张大小 | **A4** (210×297mm) | 硬编码 |
| 上边距 | **37mm** | 可配置 |
| 下边距 | **35mm** | 可配置 |
| 左边距 | **28mm** | 可配置 |
| 右边距 | **26mm** | 可配置 |
| 每行字数 | **28 字** | 可配置，通过字符间距自动计算 |
| 每页行数 | **22 行** | 可配置，通过行网格节距自动计算 |
| 文档网格 | **指定行和字符网格** | `linesAndChars` |

### 二、字体 & 字号

| 层级 | 字体 | 字号 | 加粗 |
|---|---|---|---|
| H1（一级标题） | 方正小标宋简体 | 22pt | ✅ |
| H2（二级标题） | 方正黑体简体 | 16pt | ✅ |
| H3（三级标题） | 方正楷体简体 | 16pt | ✅ |
| H4（四级标题） | 方正楷体简体 | 16pt | ✅ |
| 正文 | 方正仿宋简体 | 16pt | ❌ |
| 表格 | 宋体 | 10.5pt（五号） | 表头加粗 |
| 代码块 | Consolas | 10.5pt | ❌ |

### 三、段落格式

| 规则 | 值 |
|---|---|
| 正文/标题对齐 | 正文和 H2-H4 **两端对齐**，H1 **居中**，表格内容 **居中** |
| 行间距 | **28pt 固定值**（`exact`），不分页不浮动 |
| 段前 / 段后 | **0** |
| 首行缩进 | **2 字符**（正文 + H2-H4），代码块和表格不缩进 |

### 四、表格

- 字体号数：宋体 10.5pt，行距 14pt 固定值
- 列宽：**等分**内容区宽度，宽度类型为百分比
- 边框：**黑色单线**，1/8pt
- 表头：第一行或标记为 `tableHeader` 的单元格自动加粗
- 支持：`colspan` / `rowspan` 合并单元格

### 五、图片

- 自动缩放至**内容区宽度**，按原比例调整高度
- 来源优先级：`localPath` > `src` > `href`
- 支持格式：PNG / JPG / GIF / BMP / WebP / SVG
- WebP / SVG 自动通过 Canvas 转 PNG
- 图片居中，本地路径自动规范化（去 `file://`，URL 解码）
- 加载失败时回退为灰色斜体占位文本 `[图片: alt]`

### 六、代码块

- 字体 Consolas 10.5pt，背景色 `#F5F5F5`，行距 240 twip `auto`，不缩进

### 七、内联标记 (Marks)

| Mark | 导出效果 |
|---|---|
| `bold` | **粗体** |
| `italic` | *斜体* |
| `underline` | 下划线 |
| `strike` | 删除线 |
| `code` | 字体切换为 Consolas |
| `superscript` | 上标 |
| `subscript` | 下标 |
| `link` | 超链接样式 |
| `color` / `fontSize` | 自定义颜色/字号 |
| `highlight` | ❌ **忽略**（Word 原生不支持） |

### 八、XML 后处理（直接修改 .docx 内部 XML）

所有后处理均在 `exportWord.ts` 的 `postProcessDocx()` 中完成，用 JSZip 直接改 XML：

1. **禁用网格对齐** — 在样式和文档中插入 `snapToGrid=0`
2. **禁用孤行控制** — 全局插入 `widowControl=0`
3. **禁止标点溢出** — 全局插入 `overflowPunct=0`
4. **表格内禁用网格对齐** — 插入 `doNotSnapToGridInCell`
5. **首行缩进改用字符单位** — `firstLine`(twip) → `firstLineChars`(字符×100)
6. **表格单元格缩进归零** — 所有 `<w:tc>` 内的 `firstLine` / `left` 置零
7. **添加 Normal 样式** — 首行缩进和左侧缩进均为 0

### 九、页脚

- 居中页码，字号 12pt，字体与正文相同

### 十、旧格式兼容

如果内容不是 ProseMirror JSON（纯文本/旧 Markdown），`textToDocJson.ts` 会先预处理转换：
- `# Title` → heading, `- item` → bulletList, `**bold**` → bold mark
- 连续同类列表自动合并为单个列表节点
