# 中国石油公文写作 Skill — 阶段1：Interview 问答采集

> 适用对象：中国石油各级单位（集团公司、专业公司、地区公司、二级单位）
> 使用说明：每个文种首问区分具体场景，后续问题覆盖共性要求与中国石油特色，确保提纲生成符合央企规范与行业实际。

---

## 一、述职报告

| 序号 | 采集问题 |
|:---|:---|
| 1 | 这是什么类型的述职？（个人年度/任期/试用期/班子/党建/专项工作/安全环保） |
| 2 | 述职人姓名、职务、职级及分管的具体业务板块？（勘探开发/炼化销售/管道/新能源/工程技术/机关职能） |
| 3 | 述职时间范围？（如：202X年度、202X年X月至X月） |
| 4 | 述职场合和听众？（年度考核会/职代会/上级检查/党组织会议/安全述职） |
| 5 | 本周期核心生产经营指标完成情况？（产量、销量、效益、成本、投资等，请尽量量化） |
| 6 | 在提质增效、亏损治理、降本控费方面有哪些具体举措和成效？ |
| 7 | 最突出的亮点或创新举措？（如技术攻关、管理创新、市场开拓、数字化转型） |
| 8 | 安全环保（HSE）责任履行情况？（事故、隐患、环保指标、体系审核） |
| 9 | 党建与生产经营融合情况？（"三基本"建设、主题教育、党纪学习教育、巡察整改） |
| 10 | 廉洁自律和"一岗双责"履行情况？ |
| 11 | 存在的主要不足或短板？（建议2-3条，含原因分析） |
| 12 | 下一步改进措施和重点计划？ |
| 13 | 是否有上级最新精神、集团公司工作会议部署或考核指标需要重点呼应？ |

| 15 | 完成时限和格式要求？（是否有模板、字数限制、密级要求） |
